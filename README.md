## Installation
Use [sdmw](https://github.com/p1486/sdmw):
```
git clone https://github.com/p1486/dotfiles/tree/main
cd dotfiles
sdmw install
```
Or use `scripts/install.py`
```
git clone https://github.com/p1486/dotfiles/tree/main
cd dotfiles
python scripts/install.py install
```
